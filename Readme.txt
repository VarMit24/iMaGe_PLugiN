IMVARGE Plugin is developed by Coffee Mug Softwares. 

IMVARGE is a js plugin which can be used in any web page to give a nice and animated effect on three set of images.

Plugin can be used by developer by just calling a function imvarge().

imvarge() function requries an array as a parameter. 

In array developer can give three images locations and the width and height of div.
